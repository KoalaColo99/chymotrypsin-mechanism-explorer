# Experimental coordinate files

These mmCIF files were downloaded from the RCSB Protein Data Bank and are bundled
locally so the explorer does not depend on a remote structure service at runtime.

- `4cha.cif` — free α-chymotrypsin, 1.68 Å
- `8gch.cif` — oligopeptide-associated γ-chymotrypsin, 1.60 Å
- `1gct.cif` — acyl-enzyme candidate, 1.60 Å
- `1gg6.cif` — N-acetyl-Phe trifluoromethyl ketone transition-state analogue, 1.40 Å

See the project README and `structure-config.js` for RCSB and publication links.
